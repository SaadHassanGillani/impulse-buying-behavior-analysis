# AI-Based Recommendation Algorithms and Their Impact on Impulse Buying Behavior in Online Marketplaces

## 📌 Project Overview
This project analyzes how AI-driven recommendation systems influence impulse buying behavior in online marketplaces. It uses e-commerce event data to study user interactions such as product views, cart additions, and transactions, and applies machine learning models to predict purchase decisions.

---

## 📊 Objectives
- Analyze customer behavior in an e-commerce environment  
- Study conversion funnel (views → cart → purchase)  
- Identify patterns in user engagement and repeat interactions  
- Explore time-based shopping behavior  
- Predict purchase decisions using machine learning models  

---

## 📁 Dataset Description
The project uses multiple datasets:

- **events.csv** → User activity logs (views, cart, transactions)
- **category_tree.csv** → Product category hierarchy
- **item_properties_part1.csv & item_properties_part2.csv** → Product attributes

---

## 🔄 Data Processing Steps
- Merging and cleaning datasets  
- Removing duplicates  
- Handling missing values  
- Converting timestamps to datetime  
- Feature engineering (hour, time period, etc.)  

---

## 📈 Analysis Performed
- Event distribution analysis  
- Conversion funnel analysis  
- Repeat view behavior analysis  
- Cart behavior analysis  
- Time-based shopping behavior  
- Category-based engagement analysis  

---

## 🤖 Machine Learning Models
Two models were used to predict purchase behavior:

- Logistic Regression  
- Decision Tree Classifier  

### Evaluation Metrics:
- Accuracy  
- Precision  
- Recall  
- F1 Score  

---

## 📊 Key Insights
- Product views strongly influence purchase decisions  
- Repeat interactions increase likelihood of conversion  
- Cart additions are a strong indicator of buying intent  
- User activity varies significantly by time of day
